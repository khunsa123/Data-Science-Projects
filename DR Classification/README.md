# 🩺 Diabetic Retinopathy Classification using Machine Learning

## 📌 Project Overview
This project focuses on the application of **supervised machine learning techniques** for automated medical diagnosis, specifically the classification of **Diabetic Retinopathy (DR)** using structured features derived from retinal imaging data.

The work demonstrates how classical machine learning models can support **early disease detection** and **clinical decision-making systems**, particularly in resource-constrained environments.

---

## 🎯 Objectives
- Develop a classification framework for detecting diabetic retinopathy  
- Evaluate and compare multiple supervised learning algorithms  
- Analyze the impact of feature optimization on model performance  
- Assess computational efficiency for practical healthcare applications  

---

## 🧠 Methods & Techniques
- Supervised Machine Learning (Binary Classification)  
- Feature analysis and statistical validation  
- Hyperparameter tuning and cross-validation  

### 📊 Evaluation Metrics
- Accuracy  
- Precision  
- Recall  
- F1-score  

---

## 🤖 Models Implemented
- Support Vector Machine (SVM)  
- Random Forest  
- AdaBoost  
- K-Nearest Neighbors (KNN)  
- Gaussian Naïve Bayes  
- Gaussian Process Classifier (GPC)  
- Decision Tree  
- Artificial Neural Network (ANN)  

---

## 📊 Dataset
- **Debrecen Diabetic Retinopathy Dataset** (UCI Repository)  
- 1150 instances  
- Features represent microaneurysms and exudates extracted from retinal images  
- Binary classification: **DR vs Non-DR**

---

## 📈 Results
- Best performance achieved with:
  - **Gaussian Process Classifier (~77%)**
  - **SVM (~76.5%)**
- Feature optimization improved model stability and performance  
- Lightweight models proved effective for clinical screening scenarios  

---
