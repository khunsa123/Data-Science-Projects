# 🏠 Alberta Real Estate Analysis

## 📌 Overview

An **exploratory data analysis (EDA)** of housing prices across five major Alberta cities (Calgary, Edmonton, Red Deer, Lethbridge, Medicine Hat) covering 2023–2024. The project examines average prices by city, overall price distributions, and year-over-year price trends using structured tabular data and Seaborn visualisations.

---

## 🎯 Objectives

- Compare average housing prices across Alberta cities
- Analyse the distribution of property prices in the dataset
- Identify year-over-year price trends between 2023 and 2024

---

## 🧠 Methods

- Structured dataset creation and processing with Pandas
- Group-by aggregation for city-level and year-level summaries
- Visualisation of distributions, comparisons, and trends with Seaborn and Matplotlib

---

## 📊 Dataset

- **Type:** Structured tabular dataset (created programmatically)
- **Cities:** Calgary, Edmonton, Red Deer, Lethbridge, Medicine Hat
- **Years:** 2023–2024
- **Features:** City, Price (CAD), Bedrooms, Bathrooms, Year

---

## 📈 Outputs

| Plot | Description |
|------|-------------|
| `avg_price_city.png` | Average housing price per Alberta city |
| `price_distribution.png` | Distribution of all housing prices (KDE + histogram) |
| `price_trend.png` | Average price trend from 2023 to 2024 |

---

## 🛠️ Tech Stack

- **Language:** Python
- **Libraries:** Pandas, Matplotlib, Seaborn
