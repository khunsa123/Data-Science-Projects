# ⚡ Smart Meter Data Analysis (SAFI Project)

## 📌 Overview

This project was developed as part of my MSc dissertation in collaboration with the **SAFI (Statistical Analysis for Industry)** research group under the supervision of Dr. Amr Rashad Abdullatif and Prof. Ciprian Daniel Neagu.

The work focuses on exploratory data analysis (EDA) and clustering of large-scale smart meter data collected from households in London.

---

## ⚠️ Note on Data & Code Availability

- Due to **NDA restrictions**, only partial code is shared in this repository.  
- Core model implementation and time-series forecasting components are **not included**.  
- The dataset is large; therefore, only a small sample is provided here.  

🔗 Full Dataset: https://www.kaggle.com/jeanmidev/smart-meters-in-london

---

## 📊 Project Scope

- Exploratory Data Analysis (EDA)  
- Data preprocessing and cleaning  
- Clustering analysis on energy consumption patterns  
- Visualization of consumption trends  

---

## 🛠️ Technologies Used

- Python (Google Colab)  
- Pandas, NumPy  
- Matplotlib, Seaborn  
- Scikit-learn  

---

## 🚀 How to Run

1. Download the dataset from the link above  
2. Upload dataset to Google Drive  
3. Open the provided Colab notebooks  
4. Update dataset file paths in the code  
5. Run all cells sequentially  

---

## 📁 Repository Contents

- Colab notebooks for EDA and clustering experiments  
- Sample dataset files (limited)  
- Supporting analysis scripts  

---

## 🎯 Research Context

This project contributes to real-world industrial data analysis by exploring energy consumption behavior and demonstrating the application of data science techniques under practical constraints such as large-scale data and restricted access environments.
