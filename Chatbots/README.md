# 🤖 Simple Chatbot using Python (NLTK & Tkinter)

## 📌 Overview
This project demonstrates the implementation of a **basic chatbot in Python** using two different approaches:

1. **Rule-based chatbot with GUI** using Tkinter  
2. **NLP-based chatbot** using the NLTK library  

The chatbot can respond to simple user queries, simulate conversation, and demonstrate foundational concepts in **Natural Language Processing (NLP)** and human-computer interaction.

---

## 🎯 Project Objectives
- Build a simple conversational chatbot  
- Understand rule-based vs NLP-based chatbot design  
- Implement basic **pattern matching and response generation**  
- Explore GUI-based chatbot interaction  

---

## ⚙️ Methods Implemented

### 🧩 Method 1: Rule-Based Chatbot (Tkinter GUI)
- Built using **Tkinter** for graphical interface  
- Uses conditional statements (`if-elif`) for responses  
- Accepts user input and displays chatbot replies in a text window  

#### 💡 Features
- Simple and interactive GUI  
- Predefined responses  
- Real-time user interaction  

---

### 🧠 Method 2: NLP Chatbot (NLTK)
- Built using **NLTK’s Chat utility**  
- Uses **pattern-response pairs (regex-based matching)**  
- Includes **reflections** for conversational flow  

#### 💡 Features
- Pattern-based natural language matching  
- Dynamic responses using placeholders (e.g., `%1`)  
- Console-based interaction  

---

## 🛠️ Tools & Technologies

- **Libraries:**  
  - Tkinter (GUI development)

▶️ Method 1: GUI Chatbot
Run the Tkinter-based chatbot:

---

## ▶️ Method 2: NLTK Chatbot
Run the NLP-based chatbot:
